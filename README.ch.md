# 🖼️ 圖片探索平台

## [前往網站](https://jackyfang-image-gallery-website.netlify.app/)

## 介紹

本專案是一個使用 React 開發的圖片搜尋網站，透過 Pexels API 提供大量高品質圖片，讓使用者能依照關鍵字搜尋感興趣的圖片。

首頁會隨機載入 Pexels 的精選圖片，使用者可以輸入關鍵字搜尋圖片，並透過「查看更多」功能載入更多搜尋結果。每張圖片皆會顯示攝影師資訊，並提供原始圖片連結，方便使用者開啟或下載圖片。

此外，使用者可以將喜歡的圖片加入收藏清單，收藏資料會儲存在瀏覽器的 localStorage 中，因此即使重新整理頁面，已收藏的圖片仍會保留。專案也包含首頁、收藏頁面與自訂 404 頁面，提供完整的單頁式網站瀏覽流程。

## 技術堆疊

- 前端:`JavaScript`, `HTML`, `CSS`, `React`, `React Router DOM`, `JSX`
- 部署平台: `Netlify`
- 外部 API：`Pexels API`
