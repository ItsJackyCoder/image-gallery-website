# 🖼️ 이미지 탐색 플랫폼

## [웹사이트 바로가기](https://jackyfang-image-gallery-website.netlify.app/)

## 소개

본 프로젝트는 React로 개발한 이미지 검색 웹사이트입니다. Pexels API를 활용하여 다양한 고품질 이미지를 제공하며, 사용자는 원하는 키워드를 입력해 관심 있는 이미지를 검색할 수 있습니다.

홈 화면에서는 Pexels의 추천 이미지가 무작위로 표시됩니다. 사용자는 키워드 검색을 통해 이미지를 찾을 수 있으며, '더 보기' 기능을 이용해 추가 검색 결과를 불러올 수도 있습니다. 각 이미지에는 사진작가 정보와 원본 이미지 링크가 함께 제공되어, 이미지를 확인하거나 다운로드할 수 있습니다.

또한 마음에 드는 이미지는 '즐겨찾기'에 추가할 수 있습니다. 즐겨찾기 데이터는 브라우저의 localStorage에 저장되므로, 페이지를 새로고침한 후에도 저장한 이미지를 계속 확인할 수 있습니다. 이외에도 홈, 즐겨찾기, 사용자 지정 404 페이지를 포함하여 완성도 있는 싱글 페이지 웹사이트 이용 흐름을 구현했습니다.

## 기술 스택

- 프론트엔드:`JavaScript`, `HTML`, `CSS`, `React`, `React Router DOM`, `JSX`
- 배포 플랫폼: `Netlify`
- 외부 API：`Pexels API`
